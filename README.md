# Library Management System

A simple Python project to manage library books using classes, JSON persistence, and a command-line interface (CLI).

## Features
1. Add, issue, return, and search books
2. View all books in the inventory
3.JSON file persistence
4.Logging for all actions
5.Exception handling for input and file errors

## Folder Structure
`library_manager/` - Contains `Book` and `LibraryInventory` classes
 `cli/main.py` - Menu-driven command line interface
 `tests/` - Optional unit tests
 `books.json` - JSON file for book data
 `library.log` - Log file for operations

## Usage
```bash
python cli/main.py
